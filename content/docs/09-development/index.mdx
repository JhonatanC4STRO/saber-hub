# Desarrollo

## Requisitos

- Node.js 20+
- pnpm (recomendado) o npm
- PostgreSQL 14+ local o Docker
- Git

## Setup rápido

```bash
# 1. Clonar repositorio
git clone https://github.com/tu-org/saberhub.git
cd saberhub

# 2. Instalar dependencias
pnpm install

# 3. Crear .env
cp .env.example .env
# Editar .env con tus credenciales locales

# 4. Aplicar esquema de base de datos
npx prisma db push

# 5. (Opcional) Seed de datos iniciales
npx tsx scripts/seed.ts

# 6. Iniciar servidor de desarrollo
pnpm dev
```

Aplicación disponible en `http://localhost:3000`.

## PostgreSQL con Docker

```bash
docker run -d \
  --name saberhub-db \
  -e POSTGRES_DB=saberhub \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_PASSWORD=postgres \
  -p 5432:5432 \
  postgres:16

# .env local:
# DATABASE_URL=postgresql://postgres:postgres@localhost:5432/saberhub
```

## Scripts disponibles

| Comando | Descripción |
|---------|-------------|
| `pnpm dev` | Servidor de desarrollo con hot reload |
| `pnpm build` | Build de producción |
| `pnpm start` | Servir build de producción |
| `pnpm lint` | Lint con ESLint |
| `pnpm format` | Formatear con Prettier |
| `pnpm test` | Ejecutar suite de tests |

## Prisma Studio

Interfaz visual para explorar y editar datos:

```bash
npx prisma studio
# → http://localhost:5555
```

## Estructura de trabajo recomendada

1. Crear rama: `git checkout -b feature/mi-feature`
2. Hacer cambios
3. `pnpm lint` + `pnpm test`
4. Commit y PR a `main`
