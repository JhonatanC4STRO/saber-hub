# API REST

Todos los endpoints son Route Handlers de Next.js en `app/api/`. Retornan JSON salvo los de exportación (binario).

## Autenticación

Requieren cookie `token` (JWT HttpOnly). Se establece al hacer login. Para endpoints públicos (verificar certificado, catálogo) no se requiere.

```
Cookie: token=<jwt>
```

Para endpoints de admin se verifica además `usuario.rol.nombre === 'admin'`.

## Respuestas de error comunes

| Status | Causa |
|--------|-------|
| `400` | Cuerpo o parámetros inválidos |
| `401` | Sin token o token expirado |
| `403` | Rol insuficiente |
| `404` | Recurso no encontrado |
| `409` | Conflicto (duplicado, ya inscrito, etc.) |
| `500` | Error interno del servidor |
