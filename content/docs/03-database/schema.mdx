# Esquema Prisma

Archivo: `prisma/schema.prisma`

## Usuario

```prisma
model Usuario {
  id               Int       @id @default(autoincrement())
  nombre           String
  email            String    @unique
  passwordHash     String?
  documento        String?   @unique
  telefono         String?
  imagen           String?
  rolId            Int
  institucionId    Int?
  verificado       Boolean   @default(false)
  activo           Boolean   @default(true)
  // Brute force
  intentosFallidos Int       @default(0)
  bloqueadoHasta   DateTime?
  // Actividad
  ultimoAcceso     DateTime?
  createdAt        DateTime  @default(now())
  updatedAt        DateTime  @updatedAt
}
```

## Curso

```prisma
model Curso {
  id              Int         @id @default(autoincrement())
  titulo          String
  descripcion     String?
  imagen          String?
  estado          EstadoCurso @default(borrador)
  instructorId    Int
  institucionId   Int?
  categoriaId     Int?
  // Criterios de certificación
  leccionesMin    Int?
  evalsAprobadas  Int?
  notaGlobal      Float?
  createdAt       DateTime    @default(now())
  updatedAt       DateTime    @updatedAt
}
```

## Inscripcion

```prisma
model Inscripcion {
  id               Int               @id @default(autoincrement())
  estudianteId     Int
  cursoId          Int
  estado           EstadoInscripcion @default(activo)
  progreso         Float             @default(0)
  tiempoConectado  Int               @default(0)  // segundos (heartbeat)
  ultimoAcceso     DateTime?
  fechaInscripcion DateTime          @default(now())
}
```

## CursoExterno

```prisma
model CursoExterno {
  id              Int    @id @default(autoincrement())
  titulo          String
  descripcion     String?
  fuenteNombre    String
  fuenteUrl       String
  codigoExterno   String?
  urlOriginal     String
  duracionHoras   Int?
  nivel           String?
  modalidad       String?
  estado          String  @default("pendiente") // pendiente|aprobado|rechazado
  institucionalId Int?
  cursoId         Int?    @unique  // si se vincula a un curso interno
  creadoEn        DateTime @default(now())
}
```
